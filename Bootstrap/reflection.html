Reflection 4

I’m also guilty about this UI bad practice because I tend to do some user interface that don’t have the best appearance. For example in the contrast of the background against the text, sometimes I don’t consider the font color if the will match the background, for my perspective it blends well because I love to use dark colors but not knowing that it doesn’t have a good contrast at all. Next is creating a responsive design, this is a must because not all device has the same measurement of screen. You should learn how to do a responsive design so that your work would be compatible in different kind of device. I’ve learned this feature when we have our internship and the teach me a lesson that you should always consider the benefit of all rather than just finishing the program. Next is that you should learn how to cite sources if and only if you get something that belongs to a person or individual to credit their work and as a sign of respect to the creator. Don’t copy without giving them credits. Next, learn about the consistency of the design, it’s better to be simple yet understandable then complex and good design but the user may have a hard time using your program.
